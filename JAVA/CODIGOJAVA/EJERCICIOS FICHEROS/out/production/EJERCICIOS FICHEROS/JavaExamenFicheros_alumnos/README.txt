EXAMEN PRÁCTICO - JAVA: MANEJO DE FICHEROS

INSTRUCCIONES GENERALES:
-------------------------
- Este examen es sin conexión a Internet.
- Solo puedes trabajar con los archivos dentro del proyecto.
- No modifiques los nombres de las carpetas ni de los ficheros.
- Cada ejercicio tiene asociada una clase en `src/` que deberás completar.
- Puedes ejecutar el test en `test/` para comprobar tu progreso.
- Entrega únicamente la carpeta src/ comprimida en un archivo .zip con tu nombre: src_NombreApellido.zip

EJERCICIO 1:
------------
Clase a completar: `src/ejercicio1/LectorCSV.java`
Descripción: Lee un archivo CSV de `resources/datos.csv` y muestra su contenido formateado por columnas.

EJERCICIO 2:
------------
Clase a completar: `src/ejercicio2/RenombrarArchivo.java`
Descripción: Renombra un archivo de texto `resources/info9.txt` a `resources/datos9.txt` si existe.

CRITERIOS DE EVALUACIÓN:
------------------------
- Corrección funcional (los tests deben pasar)
- Uso correcto de clases y métodos
- Gestión de excepciones
- Limpieza del código y buenas prácticas
